1,Decendant combinators

2. Child Combinators

It means select only the elements that are direct children of a parent

3.Next Sibiling combinators

It means select the element that comes immediately after another element with the same parent.

4.Subsequent-sibling combinators

Select all the element that are siblings of a specified element and come after it. All siblings after the first elements are affected as long as they share the same parents
